Windows 빌드 이식성을 보정한 공개 미리보기입니다.

### 0.1.1 변경사항

- 영문 Windows 환경에서 PowerShell 빌드 스크립트가 UTF-8 한글 오류문구를 잘못 해석하던 문제를 수정했습니다.
- 기능과 사용자 데이터 형식은 0.1.0과 같습니다.

### 포함된 기능

- 설치 없이 실행하는 Windows x64 단일 실행 파일
- 기본 브라우저에서 사용하는 한국어 로컬 UI
- 예산표·참여인력·증빙목록·집행내역 CSV 대조
- 데이터 일관성 검사 7종과 결과 CSV·JSON 저장
- 외부 전송과 텔레메트리 없는 로컬 처리

### 주의

- 익명화된 실제 RCMS 자료로 정확도 검증을 마치지 않은 공개 미리보기입니다.
- 실행 파일은 코드 서명 인증서로 서명되지 않았습니다. `SHA256SUMS.txt`로 해시를 확인하세요.
- 비공식 데이터 점검 보조도구이며 집행 적정성이나 정산 통과를 보증하지 않습니다.
